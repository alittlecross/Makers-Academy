## User stories:

```
As a user,
So I can visit a website later,
I want to create a bookmark.

As a user,
So I can see websites I want to revisit,
I want to read a list of bookmarks.

As a user,
So I can keep them up to date,
I want to edit a bookmark.

As a user,
So I can only keep websites I want to revisit,
I want to delete a bookmark.

As a user,
So I can ensure a bookmark is useful,
I want to check a URL is valid.

As a user,
So I can why I saved a website,
I want to add a comment to a bookmark.

As a user,
So I can group my bookmarks,
I want to add a tag to a bookmark.

As a user,
So I can find particular websites,
I want to filter bookmarks by tag.

As a user,
So only I can see websites I want to revisit,
I want to create an account.

As a user,
So only I can see websites I want to revisit,
I want to sign in to my account.
```
